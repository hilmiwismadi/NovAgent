Tujuan:
Membangun LLM Agent dan mengintegrasikannya ke salah satu platform: Discord atau WhatsApp (whatsapp-web.js).
Cara Pengumpulan
1. Kumpulkan tautan GitHub di link submision yang berisi:
. Kode sumber core agent + integrasi (Discord/WhatsApp).
· README.md (singkat & jelas) mencakup:
o Setup & run untuk CLI dan integrasi (perintah install & run).
o (optional) Konfigurasi token/kredensial via .env (sertakan .env.example , jangan unggah rahasia).
Demo: GIF/tangkapan layar percakapan (termasuk proses QR untuk WhatsApp).
· tests/ (unit/functional minimal 6 kasus).
. logs/ (boleh kosong) atau contoh log.
2. Kumpulkan link laporan di dalam Notion atau Medium di link submission.
3. Kumpulkan presentasi dalam format PDF di bagian upload file.
4. Sertakan link video presentasi di link submission.
Ketentuan:
1. Tema bebas.
2. Kelompok bebas maksimal 2 orang.
Aspek Penilaian
1. Latar belakang pembuatan Agent (Anda perlu menentukan agent yang dikembangkan untuk menyelesaikan masalah apa).
2. Kompleksitas agent.
3. Kejelasan laporan pada Notion atau Medium.
4. Kejelasan Presentasi baik secara langsung ataupun tidak.
